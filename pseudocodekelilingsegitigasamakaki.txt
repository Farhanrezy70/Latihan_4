Pseudocode keliling segitiga sama kaki

Algoritma :  menghitung_keliling_segitiga sama kaki
{dibaca sisi A, sisi B dan sisi C dari piranti masukan. Hitunglah keliling segitiga sama kaki}! 

Deklarasi :
sisi A, sisi B, sisi C, keliling:int 

Deskripsi :
print "masukan sisi A"
read sisi A
print "masukan sisi B"
read sisi B
print "masukan sisi C"
read sisi C
keliling = sisi A + sisi B + sisi C
print "keliling segitiga sama kaki ="
print keliling
